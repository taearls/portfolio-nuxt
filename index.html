MusicLog

MusicLog is a fullstack application that I built for my partner, who teaches private violin lessons. The idea was to provide a central repository where students could keep track of their sheet music, as well as add practice logs to hold themselves accountable between lessons. Though this was built as my final project for the Web Development Immersive course at General Assembly, I also consider this my first freelance project, because I built it around someone else's vision. For this project, the frontend was built with React, and the backend with Sinatra. Documentation for the API that powers this app can be found here.

EthTrust

EthTrust is a project currently in progress. It is my first decentralized application built on Ethereum, so it is taking me longer to implement. I am obsessed with blockchain technology, so navigating the green landscape of the tech behind it has been a thrill.

Once completed, this project will allow users to request crowdfunded loans. If the loan request is fulfilled, they will receive a set amount of ether, and have a set amount of time to pay back the loan.

For the requestor, the incentive is obvious: receive a loan without the hassle of a possibly discrimating credit check or having to pay outrageous, predatory interest rates on it.

For the loan contributers, this is slightly more complicated. My vision for this is to implement a reward system in which users who fulfill loan requests receive a monetary reward if certain conditions are met. More details on this as I progress on the project. You can view the github repository for this project here. I'm building this with Solidity, the Truffle Framework, and the Drizzle template to integrate React into my project.
